# LandingBoi
Rails landing page generator for micro-SaaS apps

## Usage
[TBU]

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'landing_boi'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install landing_boi
```
